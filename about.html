Why I Built This App

I created this To-Do List app as part of my final web development project for school. As a student, I needed a simple way to keep track of my assignments, work tasks, and personal reminders all in one place.

This project helped me practice and demonstrate my skills in HTML, CSS, and JavaScript without using any frameworks - just pure web development basics!
